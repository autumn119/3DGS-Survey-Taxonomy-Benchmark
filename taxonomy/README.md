# 3DGS 分类体系 — 方法分类表

## 分类维度

本分类体系按照以下维度对现有 3DGS 方法进行层级分类：

| 维度 | 说明 |
|------|------|
| **任务类型** | 重建、渲染、编辑、生成、驱动等 |
| **技术模块** | 初始化策略、稠密化/剪枝、外观建模、优化策略等 |
| **输入模态** | 单目/多目图像、点云、深度图、文本描述等 |
| **应用场景** | 室内/室外、静态/动态、大规模/小物体等 |
| **效率优化** | 压缩、量化、LoD、移动端部署等 |
